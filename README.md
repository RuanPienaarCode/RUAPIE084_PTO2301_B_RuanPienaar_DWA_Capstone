## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/RuanPienaarCode/RUAPIE084_PTO2301_B_RuanPienaar_DWA_Capstone.git
   ```

2. Install dependencies:

   ```bash
   cd podspace
   npm install
   ```

## Scripts
